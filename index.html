Privacy Policy

Last updated: June 1, 2026

Data Collected

This application accesses your Google Calendar and Gmail solely for the purpose of sending you email reminders about upcoming calendar events. We only read calendar event data — no event content is stored, shared, or transmitted beyond the email notification sent to you.

Data Storage

OAuth tokens are stored securely on the server and are used only to authenticate API requests. No personal data is retained beyond what is necessary to send reminders.

Data Sharing

Your data is never shared with third parties. The application uses Google's APIs solely to read your calendar and send emails to your own address.
